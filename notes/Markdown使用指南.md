# 三、文字显示

## 1.字体

语法：

·粗体：用一对 双星号 包裹
eg:**这是粗体**

·删除线：用一对 双曲号(~) 包裹
eg:~~这是删除线~~

·下划线：用一对 (<u>)标签 包裹
eg:<u>这是下划线 </u>

·斜体：用一对 单星号 包裹
eg:*这是斜体*

·高亮：用一对 mark标签 或 双等号 包裹
eg:<mark>这是高亮<mark>
     ==这也是高亮==

快捷键：

·粗体：Ctrl + B
·删除线：Shift + Alt + (键盘上方) 数字 5
·下划线：Ctrl + U
·斜体：Ctrl + I

+++

## 2.上下标

语法：

·上标：用 ^ ^ 包裹
·下标：用 ~ ~ 包裹
eg:
水   H~2~O
x的平方   x^2^

+++

## 3.特殊字符

语法:

·乘号：\ + 乘号（星号）
·反斜线: \ + \
eg:
1乘2乘3乘4                                                    1\*2\*3\*4
输入两个\                                                         \\
输入两个 \和一个*星号                                     \\*
输入两个\，两个星号，两个\和一个星号       \\**\\* 
+++

# 四 、列表

## 1.无序列表

语法：

*/+/-  +  空格 (星号 / 加号 / 减号  加 空格)

快捷键：

·Ctrl  +  Shift  +  ]
·Ctrl  +  [
·Ctrl  +  ]
eg:
1.同一级别：

+ 张三
+ 李四
- 王五
  

2.子集类：

- 一级分类
  - 二级分类
    - 三级分类
+++

## 2.有序列表

语法 ：

数字   +  .  +  空格

快捷键：

·Ctrl  +  Shift  +  [

·Ctrl  +  [

·Ctrl  +  ]

eg:

1. 第一个内容

2. 第二个内容

   1. 第一个子内容
      1. 第一个子内容的内容

   2. 第二个子内容

3. 第三个内容
+++

## 3.任务列表

语法：(可在框内勾选 )

1. - [ ]  任务一   （- + 空格 + 左中括号 + 空格 + 右中括号 + 空格 +任务名称==无勾）
2. - [x] 任务二    （ - + 空格 + 左中括号 + x + 右中括号 + 空格 +任务名称==有勾）
+++
